# Supabase Setup Instructions for Apexstores

## Issues
The application is failing because the Supabase database and storage are not properly configured. You need to:
1. Create the `orders` table (for order saving)
2. Create the `products` table (for product management)
3. Create the `reviews` table (for customer feedback)
4. Set up proper Row-Level Security (RLS) policies for all tables
5. Create and configure the `apexstores-assets` storage bucket with proper permissions

## Solution

### Step 1: Create the Orders Table

**⚠️ DO THIS FIRST** - This table is required for the checkout functionality.

Go to your Supabase Dashboard → SQL Editor and run this SQL:

```sql
CREATE TABLE orders (
  id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
  customer_name TEXT NOT NULL,
  customer_phone TEXT NOT NULL,
  product_id BIGINT REFERENCES products(id) ON DELETE SET NULL,
  quantity INT NOT NULL DEFAULT 1,
  total_price NUMERIC NOT NULL,
  status TEXT DEFAULT 'Pending',
  payment_method TEXT DEFAULT 'M-Pesa',
  note TEXT,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT now(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT now()
);

-- Create an index for faster queries
CREATE INDEX idx_orders_created_at ON orders(created_at DESC);
CREATE INDEX idx_orders_customer_phone ON orders(customer_phone);
CREATE INDEX idx_orders_product_id ON orders(product_id);

-- Enable Row Level Security
ALTER TABLE orders ENABLE ROW LEVEL SECURITY;

-- Create RLS policies (allow all for now - adjust security as needed)
CREATE POLICY "Allow public insert" ON orders
  FOR INSERT WITH CHECK (true);

CREATE POLICY "Allow public read" ON orders
  FOR SELECT USING (true);

CREATE POLICY "Allow public update" ON orders
  FOR UPDATE USING (true) WITH CHECK (true);

CREATE POLICY "Allow public delete" ON orders
  FOR DELETE USING (true);
```

### Step 2: Create the Products Table

Go to your Supabase Dashboard → SQL Editor and run this SQL:

```sql
CREATE TABLE products (
  id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
  name TEXT NOT NULL,
  price NUMERIC NOT NULL,
  old_price NUMERIC,
  description TEXT,
  image_url TEXT,
  category TEXT DEFAULT 'electronics',
  sizes TEXT[] DEFAULT ARRAY['Standard'],
  stock INT DEFAULT 0,
  is_featured BOOLEAN DEFAULT false,
  is_new BOOLEAN DEFAULT true,
  rating NUMERIC DEFAULT 5.0,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT now(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT now()
);

-- Create an index for faster queries
CREATE INDEX idx_products_created_at ON products(created_at DESC);

-- Enable Row Level Security
ALTER TABLE products ENABLE ROW LEVEL SECURITY;

-- Create RLS policies to allow public read, authenticated insert
CREATE POLICY "Allow public read" ON products
  FOR SELECT USING (true);

CREATE POLICY "Allow authenticated insert" ON products
  FOR INSERT WITH CHECK (true);

CREATE POLICY "Allow authenticated update" ON products
  FOR UPDATE USING (true) WITH CHECK (true);

CREATE POLICY "Allow authenticated delete" ON products
  FOR DELETE USING (true);
```

### Step 3: Create the Reviews Table

Go to your Supabase Dashboard → SQL Editor and run this SQL:

```sql
CREATE TABLE reviews (
  id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
  product_id BIGINT REFERENCES products(id) ON DELETE CASCADE,
  customer_name TEXT NOT NULL,
  rating INT NOT NULL CHECK (rating >= 1 AND rating <= 5),
  comment TEXT,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT now()
);

-- Enable Row Level Security
ALTER TABLE reviews ENABLE ROW LEVEL SECURITY;

-- Create RLS policies
CREATE POLICY "Allow public read" ON reviews
  FOR SELECT USING (true);

CREATE POLICY "Allow public insert" ON reviews
  FOR INSERT WITH CHECK (true);
```

### Step 4: Create the Storage Bucket

In Supabase Dashboard → Storage:

1. Click "Create a new bucket"
2. Name it: `apexstores-assets`
3. Make it **Public**
4. Click "Create bucket"

### Step 5: Set Storage Bucket Policies

Go to the `apexstores-assets` bucket → Policies and set:

**For Public Read Access:**
```sql
CREATE POLICY "Public read" ON storage.objects
  FOR SELECT USING (bucket_id = 'apexstores-assets');
```

**For Authenticated Upload:**
```sql
CREATE POLICY "Authenticated insert" ON storage.objects
  FOR INSERT WITH CHECK (bucket_id = 'apexstores-assets');
```

**For Authenticated Delete:**
```sql
CREATE POLICY "Authenticated delete" ON storage.objects
  FOR DELETE USING (bucket_id = 'apexstores-assets');
```
